Linux二进制安装Gogs
  1.首先呢emmm,Gogs是一款极易搭建的Git服务，很适合用来做团队合作,也很适合划水ovo
  2.1）创建git用户以安全运行—–sudo adduser git(会提示设置密码，填写信息可以全部默认跳过
   2）下载二进制压缩包（看自己所需的下载，地址https://github.com/gogs/gogs/releases/tag/v0.11.66

su git
//切换至git用户
cd
//定位到git用户目录，一般是/home/git/
wget https://dl.gogs.io/0.11.66/gogs_0.11.66_linux_amd64.zip
//下载Linux最新的版本
unzip gogs_0.11.66_linux_amd64.zip
//解压

  3.接下来就是运行了ouo

cd gogs
./gogs web

   1.这样Gogs就开始运行了，就进入localhost/install安装。

最后就是自定义配置了（想咋就能咋样
  在~/gogs/custom/conf下创建app.ini 文件，可以自定义配置
自定义配置指令网址https://gogs.io/docs/advanced/configuration_cheat_sheet（其实就是官网
