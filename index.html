创建服务器


点这里ouo
  搞这个Debian/Ubuntu服务器=-=其实问了某个大佬好多好多问题

  Debian/Ubuntu安装
###1.导入public key
apt-key adv –keyserver 你自己的key
如果错误了尝试
  apt-get install dirmngr
###2.添加Deb
//Debian 9
echo “deb http://repo.mongodb.org/apt/debian stretch/mongodb-org/4.0 main” | sudo tee /etc/apt/sources.list.d/mongodb-org-4.0.list
//Debian 8
echo “deb http://repo.mongodb.org/apt/debian jessie/mongodb-org/4.0 main” | sudo tee /etc/apt/sources.list.d/mongodb-org-4.0.list
//Ubuntu 14
echo “deb [ arch=amd64 ] https://repo.mongodb.org/apt/ubuntu trusty/mongodb-org/4.0 multiverse” | sudo tee /etc/apt/sources.list.d/mongodb-org-4.0.list
//Ubuntu 16
echo “deb [ arch=amd64,arm64 ] https://repo.mongodb.org/apt/ubuntu xenial/mongodb-org/4.0 multiverse” | sudo tee /etc/apt/sources.list.d/mongodb-org-4.0.list
//Ubuntu 18
echo “deb [ arch=amd64 ] https://repo.mongodb.org/apt/ubuntu bionic/mongodb-org/4.0 multiverse” | sudo tee /etc/apt/sources.list.d/mongodb-org-4.0.list
###3.更新apt包并将其安装
apt-get update
apt-get install -y mongodb-org
###4.这就是在里面管理的一些命令（相信都能看得懂ouo
service mongod start
service mongod stop
service mongod restart
###5.不用时卸载命令
service mongod stop
apt-get purge mongodb-org*
rm -r /var/log/mongodb
rm -r /var/lib/mongodb

制作独立博客历程


点这里ouo
  因为真的是个小白,所以从一开始就开始了百度如何搭建个人博客,跟着百度中间试过了php虚拟机最后面还是换回hexo,因为其他要买域名,后面去github搭建也是受到朋友的推荐。

##制作过程
  首先我就是跟着百度那边说明的先注册了guihub账号和仓库，再去装git，node.js.和hexo.
###1.注册账号,仓库,项目和key
  我是直接百度按照这上面创建
账号：https://jingyan.baidu.com/article/455a9950abe0ada167277864.html
仓库,项目和key：
https://www.cnblogs.com/jackyroc/p/7681938.html
###2.装git过程
  1)去git官网直接download (看自己的系统而定的)版本
  2)安装一路next就行
###3.装node.js过程
  1)和git一样直接去nodejs官网直接安装(看自己的系统而定的)版本
  2)安装也是一路next
###4.装hexo过程
(装hexo问题就出现了,用git bash那边装hexo的时候会显示node.js不存在,后面去百度搜,最后面找同学知道了要用windows带的powershall)
  1)在powershall里面输入npm install hexo -g安装hexo
  2)然后在powershall里面输入hexo int folder,再输入cd folder,最后npm install达到创建
###5.配置网页
  1)我是从百度中找到配置https://hexo.io/zh-cn/docs/configuration
(这里面配置全凭个人喜好,但注意输入后面带有一个空格,不要删除,我就是一开始删除了后面配置不出,在里面将url那一行改为自己的.例:url: https://saltedyiyi.github.io然后保存.)
(  在刚刚创建的目录下找到_config.yml想要配置时发现用window自带的打不开,便再一次询问大佬问到了notepad++,用这个能够打开)
###6.创建文章
  1)在powershall里面输入hexo new post “名字”(名字可自定)
  2)再输入hexo generate生成静态文件
  3)然后输入hexo s(接着会提示一个localhost:4000用浏览器打开,若是打开了就ojbk)
###7.连接github
  1)打开git bash输入ssh -T测试链接(最后要输入一个yes)
  2)输入git config –global user.name “ “.(引号里面输入自己的github名字)
  3)再输入git config –-global user.email “ “.)(引号内输入自己注册时用的邮箱)
